# -Color_Recognition_OpenCV
# مشروع التعرف على الألوان باستخدام OpenCV

## 📌 وصف المشروع
هذا المشروع يقوم بالتعرف على لون محدد (اللون الأزرق) داخل الصور وعزله عن باقي الألوان باستخدام مكتبة **OpenCV** ولغة البرمجة **Python**. 

## ⚙️ بيئة العمل والأدوات
* **اللغة الأساسية:** Python
* **المكتبات المستخدمة:** OpenCV (`cv2`), NumPy (`np`)
* **بيئة التشغيل المقترحة:** Anaconda مع بيئة عمل Visual Studio Code (أو أي بيئة سحابية تدعم بايثون).

## 🚀 شرح خطوات عمل الكود
1. **قراءة الصورة:** يتم إدخال الصورة المراد تحليلها للمشروع.
2. **التحويل إلى نظام HSV:** تحويل ألوان الصورة من نظام BGR التقليدي إلى نظام HSV، لأن تحديد درجات الألوان فيه أدق وأسهل برمجياً.
3. **تحديد نطاق اللون:** وضع الحدود الدنيا والعليا للدرجة اللونية المطلوبة (اللون الأزرق).
4. **إنشاء القناع (Mask):** عزل الأجزاء التي تطابق اللون الأزرق وتغطية باقي الأجزاء.
5. **استخراج النتيجة وحفظها:** دمج القناع مع الصورة الأصلية لحفظ الصورة النهائية وإظهار اللون المحدد فقط.

## 💻 كيفية التشغيل
1. تأكد من تثبيت المكتبات عبر الأمر:
   `pip install opencv-python numpy`
2. ضع صورة باسم `image.jpg` في نفس مجلد المشروع.
3. قم بتشغيل الملف البرمجي:
   `python color_recognition.py`
4. ستظهر لك النتيجة محفوظة باسم `detected_color.jpg`.
ولقد تم كتابة كود python في ملف منفصل لتطبيق المهمه 🩶
